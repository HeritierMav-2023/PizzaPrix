# PizzaPrix
Points d’attention
Certains ingrédients (comme les pommes de terre) manquaient dans le tableau initial ; ils ont été ajoutés avec un prix estimé.

Les noms des ingrédients doivent correspondre exactement à ceux utilisés dans les recettes (respect de la casse et des accents). La recherche par nom est insensible à la casse pour faciliter la saisie.
